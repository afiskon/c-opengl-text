# cpp-opengl-models

Usage:

```
    git submodule init
    git submodule update
    mkdir build
    cd build
    cmake ..
    make
    cd ..
    ./build/emdconv models/skybox.blend skybox.emd
    ./build/models
```

* WASD + mouse - move camera
* Z/X - enable/disable wireframes mode
* Q - quit

Tested on Linux, MacOS and Windows.
